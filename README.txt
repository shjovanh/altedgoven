# Dorfgemeinschaft Alt-Edgoven e. V.

Dies ist die offizielle Website der Dorfgemeinschaft Alt-Edgoven e. V., bereitgestellt über GitHub Pages.

## Inhalte
- Vereinsvorstellung
- Termine & Aktivitäten
- Kontaktinformationen
- Impressum & Datenschutz

## Hosting
Die Website wird statisch über GitHub Pages betrieben.

## Quellen & Lizenzen
Das Design basiert auf dem Template **Hyperspace** von [HTML5UP](https://html5up.net/) unter der CC BY 3.0 Lizenz.
